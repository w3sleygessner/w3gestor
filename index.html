Como funcionam os Templates e Variações

1. Variáveis Mágicas (Automáticas)

Estas palavras são substituídas instantaneamente pelos dados individuais de cada cliente no momento do envio:

{cliente} - Nome do cliente
{app} - Nome do Aplicativo
{plano} - Nome do Plano
{dns} - URL/DNS do App
{vencimento} - Data de Vencimento
{valor} - Valor do Plano
{usuario} - Usuário de Acesso
{senha} - Senha de Acesso
{dias} - Dias Restantes

2. Variação de Texto Anti-Ban (Spintax)

Para evitar que o WhatsApp detete padrões robóticos, use chavetas com palavras separadas por uma barra vertical |. O sistema escolherá uma opção aleatória para cada mensagem enviado.

{Olá|Bom dia|Fala} {cliente}, {sua assinatura|seu acesso} do app {app} {vence|expira} dia {vencimento}.

Resultado no Cliente 1: Olá Wesley, sua assinatura do app P2P vence dia 25/05/2026.

Resultado no Cliente 2: Bom dia João, seu acesso do app P2P expira dia 25/05/2026.

Padrão do Comprovante

Este texto será usado como base para os envios automáticos ao dar baixa numa fatura. Pode usar as variáveis mágicas (Spintax suportado).
